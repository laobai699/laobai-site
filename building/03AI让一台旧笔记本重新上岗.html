鸟枪换炮
AI指导我安装和使用软件

建站初期，我是在手机上和 AI 聊天、搭建最基本的骨架的。很快就感觉到这也太低效了——光是复制粘贴，在手机上操作起来就无法精准定位。

我清楚地意识到：还需要一套顺手、稳定、能长期使用的工具。

于是，这台老伙计重新上岗了

机龄快10年，硬盘238G已经使用了237G，眨眼都困难的老电脑——就这样被重新翻了出来。

要重新安装操作系统，对已经十年不怎么碰电脑的我来说，如果没有 AI 的帮助，这已经是不可能完成的任务了。

建站工具大师

帮我安装和配置这台电脑的 AI，我给他起了个名字：建站工具大师。他来自 ChatGPT。在之后的建站过程中，所有软硬件方面的问题，我都去请教他。

在与他的交流中，我慢慢了解到需要什么样的软件。他一件一件地引导我安装好了这套工具：

Chrome
主浏览器，用来查看网页效果
这个我知道，打开网站用的
VS Code
主要代码编辑器
闻所未闻
Live Server
本地预览网页，手机平板同步查看效果
闻所未闻
GitHub Desktop
同步网站仓库，管理版本更新
闻所未闻
FileZilla
为以后连接传统服务器做准备
闻所未闻
不懂没关系。
AI 会告诉你装什么、怎么装。
AI 还告诉你怎么用。
AI 已将美味大餐奉上，我只需要张开嘴巴了。

边学边用，VS Code 成为了画板，千奇百怪的代码在这里编辑，植入到里面的Live Server成为了眼睛，实时观测着页面的变化。GitHub Desktop是邮递员，负责把我电脑端的网页转递到云端。

一台差点被淘汰的旧机器，就这样重新上岗了。
