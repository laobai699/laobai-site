3D 打印笔记
NO. 01 · FDM 打印机
Creality Ender 3 Pro · 操作指南
从有模型文件到打印完成的完整流程。包括耗材更换、打印床调平、Cura 切片设置，以及常见问题处理。适合间隔较久重新上手时查阅。
PLA 耗材 手动调平 Cura 切片 PEI 磁吸板
一、开始前的检查

每次打印前花两分钟做基础检查，可以避免大多数失败。

  • PEI 磁吸钢板是否清洁——用酒精擦拭，不要用手触摸表面，手上油脂会严重影响粘附。
  • 耗材是否正常安装，料盘挂在机器右侧料架上，耗材穿过导料管。
  • 耗材是否受潮——正常料有韧性，弯曲不断；受潮料硬邦邦，一弯即折，建议直接换新料。
  • SD 卡是否插入右侧插槽。
PEI 板的清洁是第一层能否粘住的关键，每次打印前都要做。酒精擦完自然晾干再开始。
二、更换耗材

如果耗材已经装好且颜色合适，可以跳过这一步直接去第三步调平。

退出旧料

  1. 菜单操作：Control → Temperature → Nozzle，设置 200°C，等待加热完成。
  2. 菜单操作：Prepare → Move Axis → Move Extruder。
  3. 选择步进 10mm，向负方向转动旋钮退料，同时用手轻轻向上拉耗材。
  4. 持续操作直到耗材从进料口完全退出。

装入新料

  1. 新料挂上右侧料架，用剪刀将料头剪成 45 度斜角，方便插入。
  2. 将耗材穿过导料管，推到进料齿轮处。
  3. Move Extruder 向正方向转动旋钮送料。
  4. 持续送料，直到喷嘴挤出颜色均匀的新料（约 10–15cm），换料完成。
Move Extruder 10mm 指的是送入耗材 10mm，实际从喷嘴挤出长度会更长。
三、打印床调平

调平是 FDM 打印成功的核心。间隔较久重新使用，或者更换耗材后，都建议重新调平一次。

准备工作

  1. 预热：热床 60°C，喷嘴 200°C。必须在加热状态下调平，冷热会影响实际间距。
  2. 菜单：Prepare → Auto Home，等打印头归零停稳。
  3. 准备一张普通 A4 打印纸备用。

调节四个角 + 中心

每次换角之前，先升高 Z 轴 10mm 再移动，避免喷嘴刮伤打印床。

左前角
X=10，Y=10。降下 Z 轴，放入 A4 纸，调节左前蝴蝶螺母。
右前角
X=220，Y=10。升 Z → 移动 → 降 Z，调节右前蝴蝶螺母。
左后角
X=10，Y=220。升 Z → 移动 → 降 Z，调节左后蝴蝶螺母。
右后角
X=220，Y=220。升 Z → 移动 → 降 Z，调节右后蝴蝶螺母。
中心
X=110，Y=110。只做验证，无螺母可调。中间稍微松一点点是正常的。

纸张手感判断

  • 纸能滑动，有轻微摩擦感——正确，继续下一个点。
  • 纸完全卡住，抽不动——太近，顺时针拧螺母，让床面下降。
  • 纸完全没有阻力——太远，逆时针拧螺母，让床面上升。
移动安全原则：每次换到下一个角，先升高 Z 轴 10mm，再移动 X/Y，再降回去。这样喷嘴不会刮到床面。
四、Ultimaker Cura 切片

切片就是把 3D 模型文件转换成打印机能读懂的指令文件。Cura 是免费软件，Ender 3 Pro 有现成预设，开箱即用。

初次安装配置（只做一次）

  1. 下载地址：ultimaker.com/software/ultimaker-cura，安装时一路 Next。
  2. 启动后弹出登录页，直接点 Skip 跳过，无需账号。
  3. 添加打印机：Non UltiMaker Printer → Creality → Ender-3 Pro → Add。
  4. Machine Settings 页面不需要修改任何参数，直接确认完成。

导入模型

  1. 将 STL 文件直接拖入 Cura 的打印床区域，或菜单 File → Open File(s)。
  2. 用鼠标中键拖动旋转视角，滚轮缩放，右键平移。
  3. 点击模型，左侧工具栏可以移动、旋转、缩放。
  4. 多个模型：右键空白处 → Arrange All Models，自动排列到合理位置。
关于模型方向：让最大的平面朝下放置，可以减少支撑用量，提高成功率。左侧旋转工具可以调整角度。

关键参数设置

参数 推荐值 说明
Layer Height 层高 0.2mm 标准质量，速度与细节的平衡点
Infill Density 填充率 15–20% 摆件、雕像不需要高填充
Support 支撑 按需开启 文件名含 "supported" 的模型务必关闭！
Support Structure Tree（树形） 异形复杂模型首选，打完后更容易去除
Adhesion 附着 Brim（边沿） 在模型底部加一圈，提升第一层粘附力
重要：如果下载的 STL 文件名含有 supported，说明模型已经内置了支撑结构。这时 Cura 里的 Support 必须关闭，否则双重支撑会让打印失败。

导出文件

  1. 点击右下角 Slice 按钮，等待切片完成。
  2. 查看右下角显示的预计打印时间和耗材用量，判断是否合理。
  3. 点击 Save to Disk，将文件保存到 SD 卡根目录。
五、开始打印
  1. 将 SD 卡插入打印机右侧插槽。
  2. 菜单：Print from SD → 找到文件 → 确认开始。
  3. 打印开始后，全程观察前 5 分钟的第一层情况。

第一层判断

✅ 正常
耗材均匀、被轻微压扁，紧贴粘在床面上。继续观察即可。
喷嘴太远
耗材像面条一样堆在床面上，不粘。菜单找 Z Offset 或 Babystep Z，向负方向微调 -0.1mm，观察变化。
喷嘴太近
喷嘴刮着床面，出料困难，有摩擦声。向正方向微调 +0.1mm。
不粘床
耗材完全不粘，直接翘起来。暂停打印，重新酒精清洁 PEI 板，检查调平。
Z Offset 微调位置：打印过程中可以进入菜单 Control → Motion → Z Offset（部分固件显示为 Babystep Z），不需要停止打印就能实时调整。
六、打印后处理
  1. 打印完成后，等待打印床冷却。PEI 板冷却后模型会自然松动，不要趁热强掰。
  2. 取下磁吸 PEI 钢板，轻轻弯曲钢板，模型会自然脱落。
  3. 用剪钳或尖嘴钳小心去除支撑结构，从支撑根部剪断，不要硬拉。
  4. 用砂纸打磨支撑痕迹，从 400 目到 1000 目逐步细化，直到满意。
  5. 如需上色，丙烯酸颜料上手。想要效果好，可以先喷一层灰色底漆。
七、常见问题
第一层不粘
最常见。先酒精清洁 PEI 板，再检查调平，再用 Z Offset 微调喷嘴高度。三步按顺序来。
中途脱落
调平不准，或打印速度太快。下次打印前重新调平，Cura 里适当降低打印速度。
耗材不出料
堵头或耗材受潮。先加热到 200°C 后用 Move Extruder 尝试疏通，无效则换新料。
层错位
打印速度太快或皮带松动。Cura 里降低速度，或检查 X/Y 轴皮带张力。
翘边
热床温度不够或床面不够干净。确保热床达到 60°C，重新酒精清洁 PEI 板。
表面拉丝
移动过程中漏料。Cura 里调整 Retraction（回抽）参数，或将打印温度降低 5°C 试试。
