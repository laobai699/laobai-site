🏠 家庭网络安全“防身术”

享受智能生活，更要守护数字家园

两个让您家里网络更安全的小秘密。简单几步，就能防止“隔墙有耳”。

1. 您知道家里有两个“门卫”吗？

大多数人只改了第一个，却忘了第二个：

  • Wi-Fi 密码： 决定谁能连上网络（像家门钥匙）。
  • 管理员密码： 决定谁能更改路由器设置（像保险箱钥匙）。

⚠️ 警告：如果管理员密码保持初始（如 admin/password），任何连上 Wi-Fi 的人都能控制您的路由器！

2. 为什么要改“管理员密码”？

如果坏人进了您的路由器后台，他们可以：

  • 改掉 Wi-Fi 密码，让您全家断网。
  • 偷看您家里连接了哪些设备（如监控、电脑）。
  • 甚至植入后门，长期监控您的上网记录。

3. 神器：开启“访客网络” (Guest Wi-Fi)

当有客人（包括像我这样来服务的）需要联网时，请给他们专用的 Guest Wi-Fi：

  • 物理隔离： 客人只能上网，看不见您家里的私密电脑和监控。
  • 保护后台： 连访客网络的人无法打开路由器管理页面。
  • 自动限速： 不会因为客人在看视频而影响您的办公速度。

4. 简单三步走：

  1. 浏览器输入 10.0.0.1 (Rogers) 或 192.168.1.254 (Telus)。
  2. 找到 Security 或 Admin，修改初始管理员密码。
  3. 开启 Guest Network 功能，设置一个独立的访客名字。
